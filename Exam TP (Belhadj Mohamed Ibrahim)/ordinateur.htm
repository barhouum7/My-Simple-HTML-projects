Ordinateur

Un Ordimateur est un ensemble de circuits electroniques permettant de manipuler des donnees sous forme binaire.

Il existe plusieures types :

  • Oridinateur Portable
  • Ordinateur de Bureau
  • Ordinateur de Poche

L'utilisation d'ordinateur de poche augmente vue l'apparition de reseau sans fils voir le tableau qui montre cette augmentation.
